Text all in One

一款在 iOS 和 iPadOS 上进行文本处理的现代化应用，为剪贴板、Markdown 编辑和文本工作流提供无缝的体验。

我们正在积极开发 macOS 版本，它将很快成为一个支持所有 Apple 平台的通用应用。

剪贴板

Taio 将剪贴板作为输入源之一，可以快速抓取剪贴板临时使用。保存的剪贴板可以使用 iCloud 跨设备同步。

  • 快速捕捉
  • 文本处理
  • iCloud 同步

编辑器

Taio 提供了一个成熟的 Markdown 编辑器，支持 CommonMark 和 GitHub Flavored 标准。文件系统是开放的，可以被其他编辑器访问。

  • 开放文件系统
  • 扩展系统
  • 100% CommonMark 支持

文本动作

一个为剪贴板和编辑器提供可扩展性的机制，工作原理类似于 快捷指令 应用。对于复杂的场景，您甚至可以借助 JavaScript 编写动作。

  • 文本工作流
  • 可视化编辑器
  • 基于 JSBox 的 JS 环境

更多特性

原生的界面

我们使用原生组件以获得最佳的用户体验，尽可能地利用平台功能。如果您喜欢使用 iOS 应用，您应该会倍感舒适。

为 iPad 优化

与把 iPhone 应用放大不同的是，我们把 iPad 视为一等公民。支持分屏或多窗口等功能，也关心硬件外设的体验。

多窗口编辑

同时编辑和预览多个文件，支持多窗口，让编辑更灵活，处理任务更容易。

键盘和触控板

全面支持键盘快捷键和触控板交互，在连接外部硬件时，可获得桌面级体验。

文件模板

不必每次都从零开始创建文件，可以使用模板来加速。文件模板支持动态内容，如剪贴板和日期。

无障碍环境

优秀的产品应该是所有人都能使用的，我们会尽最大努力实现这一目标。欢迎对无障碍问题进行反馈，改进是我们的责任。
